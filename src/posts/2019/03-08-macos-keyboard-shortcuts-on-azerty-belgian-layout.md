cheatsheet to keep some shortcuts I am always looking at on mac when coding on macosx. 

> tilde (~) => [ **ALT + N** ]  (alt + n)

> pipe (|) => [ **SHIFT + ALT + L** ]

> open bracket ({) [ **ALT + (** ]

> closing bracket (}) [ **ALT + )** ] 

> open array ([) [ **SHIFT + ALT + (** ]

> closing array (]) [ **SHIFT + ALT + )** ]

> less or equal (≤) [ **ALT + <** 

> greater or equal (≤) [ **ALT + SHIFT + >** ]
